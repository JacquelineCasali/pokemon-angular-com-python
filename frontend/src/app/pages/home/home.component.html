Bem-vindo à API Pokédex

Descubra o mundo Pokémon com nossa abrangente API Pokédex, meticulosamente elaborada usando FastAPI e Angular. Mergulhe em informações detalhadas sobre cada Pokémon, incluindo tipos, poderes e muito mais.
